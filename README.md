# 🌿 肠道日记 · 排便健康记录小程序

> 每天一记，读懂你的肚子。

一款面向年轻人的肠道健康自我管理微信小程序，帮助用户科学记录每日排便状况，通过数据可视化和健康分析，提升对肠道健康的关注意识，培养规律的排便习惯。

---

## ✨ 功能特性

### 快速记录
- 首页悬浮按钮（FAB），3 步完成一次排便记录
- 基于**布里斯托大便分类法**（Type 1–7）的图文形态评分
- 支持记录排便时间、时长、颜色、感受标签及自由备注
- 自动读取当前时间，支持手动修改

### 统计分析
- 日 / 周 / 月频次柱状图
- 布里斯托评分趋势折线图
- 规律性分析（平均排便时间段、平均间隔）
- 自动生成健康评估标签：规律 / 偏少 / 偏多 / 需关注

### 历史记录
- 日历视图，有记录日期绿点高亮
- 列表视图按日期分组展示
- 支持编辑、删除已有记录

### 提醒与习惯养成
- 用户自定义每日提醒时间
- 首页展示连续打卡天数，激励坚持
- 当天未记录时晚间推送提醒

### 数据管理
- 微信登录后云端自动同步
- 支持导出 PDF 健康报告（供就医参考）

## 🚀 本地运行

1. 克隆本仓库

```bash
git clone https://github.com/你的用户名/intestine-diary.git
```

2. 打开**微信开发者工具**，选择「导入项目」

3. 填入你的小程序 AppID（需在[微信公众平台](https://mp.weixin.qq.com)注册）

4. 点击「编译」即可预览
